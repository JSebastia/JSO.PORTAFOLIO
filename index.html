Proyectos destacados

  • Amigo Secreto

    Una web donde podrás ingresar el nombre de tus amigos y sortearlos aleatoriamente para una determinada ocasión. ¡Corre y sortea los nombres de tus amigos!

    Tecnologias utilizadas:

    HTML5

    CSS3

    JavaScript

    GitHub

  • Conversor de monedas

    Una aplicación de consola donde puedes ingresar un tipo de moneda y escoger a que moneda deseas la conversión. Excelente para saber la equivalencia de tu moneda en diferentes países en tiempo real.

    Tecnologias utilizadas:

    Java

    Spring

    SQL

    API

    GitHub

  • LiterAlura

    Una aplicación de consola donde puedes ingresar el nombre de tu libro favorito y encontrar las características del mismo, y muchas funciones más. ¿Qué esperas? ¡Anímate, y busca tu libro!

    Tecnologias utilizadas:

    Java

    Spring

    SQL

    API

    GitHub

  • ForoHub

    Una API Rest que simula un Foro. Con ella podrás ver las funcionalidades que tiene un Foro sencillo. Puedes crear, comentar y editar en el Foro. ¡Anímate, y da un paso a utilizar la API y descubre la magia!

    Tecnologias utilizadas:

    Java

    Spring Boot

    Spring Security

    MySQL

    GitHub

Sobre mí

Mis redes:

GitHub LinKedin

Desde pequeño siempre me ha llamado la atención las diferentes tecnologías que ayudaban y servían a las personas. Siempre me hacía muchas preguntas; ¿Cómo funcionan las redes sociales?, ¿Qué hay detrás del videojuego que juego todos los días? y ¿Cómo funciona el internet? Y entre otras preguntas que me hacían despertar curiosidad sobre el funcionamiento de las cosas. De allí nació una pasión hacia la tecnología, la innovación, la programación y las telecomunicaciones.

Actualmente estoy estudiando Ingeniería de Telecomunicaciones en la Universidad de Antioquia. Tengo un extenso conocimiento en varias tecnologías y herramientas, donde he trabajado en algunos proyectos prácticos.

Busco seguir alimentando esa curiosidad y pasión que alguna vez me empujaron a aprender, por lo que constantemente aprendo cosas nuevas. Imagino un mundo sostenible y en paz, donde todas las personas y el ambiente estén en armonía gracias a la tecnología y el esfuerzo de millones de mentes.
